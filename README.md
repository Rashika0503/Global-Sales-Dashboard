# Global Sales Dashboard

## Overview
An interactive Power BI dashboard built using the Global Superstore dataset to analyze sales, profit, customer segments, market performance, and business trends.

## Tools Used
- Power BI
- Global Superstore Dataset

## Features
- KPI Cards
- Interactive Filters
- Time-Series Analysis
- Navigation Menu
- Market & Product Insights

## Dashboard Pages
### Business Performance Overview
Provides insights into sales, profit, orders, market-wise profit, and sales trends.

### Product & Market Insights
Analyzes top-performing products, countries, quantity sold, and category-wise sales.
